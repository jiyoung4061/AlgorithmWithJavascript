### 페이지교체알고리즘(LRU)
**LRU : 교체가 자주 이뤄지는 페이지는 보존**
- 최근 페이지 교체가 이뤄진 것을 뒤로 뺀다 => 앞에 부분만 삭제해주면됨
```
array.shift() : 맨 앞에 요소 삭제
array.push(shift()) : 맨앞의 요소를 삭제한 후, 맨 뒤에 push
=> 오류 : hit된 요소가 맨앞에 없을 수도 있음.
따라서, 해당 동물의 index를 찾아서 1개 지워주는 -> splice사용
```
